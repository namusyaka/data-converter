=data-converter


==Install
  gem install data-converter

==Description
  Convert hash and array into HTML Element (Array => ul, Hash => dl).
  It's under the development.

==Code
  require 'rubygems'
  require 'data-converter'

  puts DataConverter.parse({
    :a => "b",
    :c => { :d => ["<>&\"'", 1, 2] }
  })
